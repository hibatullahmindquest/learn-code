# Design workflow

Whenever a task involves building a new page or any visual/design work in
this project, invoke one of the installed design skills in
`.agents/skills/` (e.g. `minimalist-ui`, `high-end-visual-design`,
`redesign-existing-projects`, `image-to-code`, `stitch-design-taste`,
`design-taste-frontend`, `gpt-taste`, `brandkit`, `industrial-brutalist-ui`,
`imagegen-frontend-web`, `imagegen-frontend-mobile`). Ask the user which
skill to use before starting, unless they've already specified one.
